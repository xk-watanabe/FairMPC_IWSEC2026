# Formal Verification of Fair Multi-Party Computation based on Financial Penalties using Smart Contracts

This is a repository for sharing ProVerif codes of the paper entitled above submitted to IWSEC 2026.
Each file corresponds to the verification cases as follows (also see Sec. 4.1):
- Flow 1: `fairness_honest.pv`
- Flow 2: `fairness_malicious.pv`
- Flow 3: `fairness_honest_delta.pv`
- Flow 4: `fairness_malicious_delta.pv`
- Flow 5: `fairness_malicious_2p.pv`
- Adversarial Interface Model: `fairness_adversarial.pv`
